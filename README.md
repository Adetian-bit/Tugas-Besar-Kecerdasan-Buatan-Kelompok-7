# Tugas-Besar-Kecerdasan-Buatan-Kelompok-7
## ▶️ Cara Menjalankan Program (Notebook)

1. Pastikan file dataset berada pada folder yang sama dengan notebook  
   (contoh: `resiko kredit.xlsx`).

2. Buka file notebook (`.ipynb`) menggunakan:
   - Jupyter Notebook
   - JupyterLab
   - Google Colab  
     > Jika menggunakan Google Colab, upload file notebook dan dataset terlebih dahulu.

3. Jalankan seluruh cell dari atas ke bawah menggunakan fitur **Run All**.

4. **Studi Kasus Risiko Kredit**  
   - Program menghitung skor menggunakan metode **Fuzzy Mamdani** dan **Fuzzy Sugeno** untuk seluruh ID.
   - Menampilkan **Top-10 risiko kredit terkecil**.
   - Menyimpan file output **`peringkat.xlsx`** jika bagian export diaktifkan.

5. **Studi Kasus Beasiswa**  
   - Program menghitung nilai kelayakan setiap mahasiswa.
   - Menampilkan tabel hasil perhitungan.
   - Menyediakan **fungsi debug** untuk melihat detail perhitungan per mahasiswa.
